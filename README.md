## 先搞清楚机房和线路的关系

DMIT 的产品逻辑其实不复杂，两个维度：**地区（机房位置）× 线路等级**。

**线路等级**基本分三档：

## 三个主要机房怎么选

### 香港机房（HKG）——延迟最低，价格最贵

### 洛杉矶机房（LAX）——产品线最全，选择最多

LAX 是 DMIT 的主力机房，套餐数量最多，也是大多数用户的第一选择。三网 CN2 GIA 回程，电信联通去程直连，移动去程 CMI，晚高峰下来延迟在 150-180ms 区间，丢包率低。

LAX.Pro 适合电信用户、对速度有刚需的。LAX.EB 走 CMIN2，联通移动用户可以优先考虑，性价比更高。如果需要高防，LAX.sPro 带 5Tbps 级 DDoS 防护，适合建站。

### 东京机房（TYO）——日本 IP、亚太低延迟

需要日本节点、或者做面向东南亚业务的首选。CN2 GIA + 9929 + CMI 三路优化，亚太地区访问体验不错。入门套餐年付 $36.9 起，是 DMIT 家价格最亲民的机房之一。

## 购买流程：注册到开机，一步步来

1. **注册账号**：进官网，填邮箱 + 密码，选择所在国家。注意：新注册账号可能有 7 天等待期才能下单，建议提前注册好。

2. **选产品**：在后台点 Create → Server，依次选地区、产品系列（Premium / Eyeball / T1）、具体套餐。

3. **选付款周期**：月付、季付、半年付、年付都有，周期越长单价越低，且优惠码通常需要季付以上才能激活。

4. **贴优惠码**：结算页面找到 "Apply Promo Code" 输入框，粘贴（不要手打，大小写敏感），点 Validate Code 验证，看到价格变化再提交订单。

5. **付款**：支付宝、微信、PayPal、信用卡都支持，国内用户用支付宝最方便。

6. **SSH 连接**：DMIT 默认是 SSH 密钥登录，不支持用密码直接连。开机后需要用私钥登录，官网有教程，新手建议先看一遍再操作。

7. **退款政策**：3天内流量使用不超过 30GB 可全额退；30天内按剩余价值退款（退至账户余额免手续费，原路返回可能扣手续费）。用不惯的话，三天内退掉成本很低。

## 常见问题

**Q：DMIT 新用户注册后能立刻下单吗？**

大多数情况可以，但部分新账号有 7 天等待期。如果你急着用，建议提前几天注册好账号，再等到要下单时操作。

**Q：优惠码用过一次以后续费还有折扣吗？**

DMIT 的多数优惠码是"循环折扣"，续费时依然按折扣价收费。但每个账号限用一次，所以第一次选准了很重要。

**Q：流量超了会停机吗？**

T1 系列超量后不停机，会自动限速到 50-100Mbps，月初流量重置后恢复正常。Pro 系列部分套餐也支持超量不断网。

**Q：DMIT 支持 Windows 系统吗？**

目前只有 Linux 系统，不提供 Windows VPS。

**Q：付款后多久可以开通？**

一般几分钟内就会分配好，官网后台刷新即可看到机器状态。

**Q：退款能退到支付宝吗？**

退款可以选择原路返回（可能扣支付网关手续费）或退至 DMIT 账户余额（免手续费）。想退到支付宝本身没问题，但手续费那部分会被扣掉。
